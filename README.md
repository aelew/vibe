## Installation

[Click here](https://github.com/aelew/vibe/releases/latest) to download the latest release.

## Run app (dev)

Clone the project

```bash
git clone https://github.com/aelew/vibe.git
```

Go to the project directory

```bash
cd vibe
```

Install dependencies

```bash
pnpm install
```

Launch Vibe

```bash
pnpm tauri dev
```
